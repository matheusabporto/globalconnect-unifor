# 🌐 GlobalConnect Unifor

> Projeto de Engenharia de Requisitos — Análise e Desenvolvimento de Sistemas | Unifor (2º semestre)

## 📋 Sobre o Projeto

Especificação de um sistema web/mobile para gestão da mobilidade acadêmica internacional da 
Universidade de Fortaleza (UNIFOR), alinhado ao programa do Núcleo de Estratégias 
Internacionais (NEI), que oferece acesso a mais de 140 instituições em cerca de 30 países.

O projeto foi desenvolvido aplicando técnicas de Engenharia de Requisitos, incluindo 
elicitação com stakeholders, modelagem UML, prototipação e especificação seguindo 
padrões IEEE 830.

## 🎯 Módulos do Sistema

- **Módulo do Estudante** — Portal de oportunidades, autoavaliação de elegibilidade, 
  portfólio digital e acompanhamento do processo seletivo
- **Módulo de Gestão (NEI)** — Gestão de convênios, painel de candidaturas, 
  notificações e relatórios
- **Módulo de Pós-Intercâmbio** — Relatos de experiência, aproveitamento de créditos 
  e rede de mentoria
- **Módulo Colaborativo** — Sistema Tandem, agenda de eventos e fórum de dúvidas
